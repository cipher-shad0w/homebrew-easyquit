# Homebrew Tap for EasyQuit

This is the official Homebrew tap for [EasyQuit](https://github.com/cipher-shad0w/EasyQuit).

## Installation

```bash
brew install cipher-shad0w/easyquit/easyquit
```

## About EasyQuit

EasyQuit is a simple and efficient application designed to help users quickly exit or close applications on their macOS computer.

### Features

- **Double-click to Quit**: Close an application gracefully with a double-click
- **Right-click Force Quit**: Context menu with "Force Quit" option for hanging apps
- **Search Bar**: Quick search to find applications
- **Lightweight**: Minimal resource consumption

## Updating

```bash
brew upgrade easyquit
```

## Uninstalling

```bash
brew uninstall easyquit
```

## Note

This app uses ad-hoc signing. On first launch, you may need to allow it in:
**System Preferences > Security & Privacy > General**
